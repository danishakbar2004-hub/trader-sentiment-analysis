# Trader Behavior vs Market Sentiment Analysis

This project analyzes how trader behavior and profitability change under different market sentiment conditions using the Bitcoin Fear & Greed Index and historical trading data.

## Dataset

The Fear & Greed Index dataset is included in this repository.

The historical trading dataset is large (>25MB) and therefore not uploaded to GitHub.
It was used locally for analysis.

## Project Structure

- `trader_sentiment_analysis.ipynb` — Main notebook containing analysis and visualizations
- `fear_greed_index.csv` — Market sentiment dataset
- `README.md` — Project documentation
